HTML HEAD AND METADATA

The <head> section stores information about the page rather than content that is directly shown in the browser window.

Important head elements include <title>, <meta>, <link>, <style>, <script>, and <base>. These elements control page title, character encoding, responsive behavior, search descriptions, external files, and scripts.

A well-written head section makes the page load correctly, display the right text, and provide better information to search engines and social platforms.

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="HTML lesson about metadata">
  <title>HTML Head Example</title>
  <link rel="icon" href="favicon.ico">
</head>
