HTML EMBEDDING AND GRAPHICS

HTML can embed external content and display graphics by using elements such as <iframe>, <embed>, <object>, <svg>, and <canvas>.

The <iframe> tag is often used to show another webpage, map, or video inside the current page. SVG is ideal for scalable icons and shapes, while canvas is commonly used for drawing with JavaScript.

These elements are usually taught together because they all deal with bringing visual or embedded content into a page beyond normal text and images.

<iframe src="https://example.com" title="Embedded Example"></iframe>
<canvas width="300" height="150"></canvas>
