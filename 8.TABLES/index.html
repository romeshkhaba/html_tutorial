HTML TABLES

Tables are used to display data in rows and columns. They are best for structured information such as schedules, marks, prices, or reports.

A table is built with <table>, rows use <tr>, headers use <th>, and standard data cells use <td>. Larger tables often use <caption>, <thead>, <tbody>, and <tfoot>.

Tables should be used for real data, not for page layout, because semantic layout tags are better for structure.
