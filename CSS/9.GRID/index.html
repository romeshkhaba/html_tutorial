CSS GRID

CSS Grid is a two-dimensional layout system. It can control both rows and columns at the same time, which makes it powerful for larger page layouts and structured content areas.

Important grid properties include display: grid, grid-template-columns, grid-template-rows, gap, grid-column, grid-row, and named grid areas.

Grid works especially well for dashboards, galleries, page sections, and any design that needs precise row and column control.
