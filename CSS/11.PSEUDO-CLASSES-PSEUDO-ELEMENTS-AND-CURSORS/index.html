CSS PSEUDO-CLASSES, PSEUDO-ELEMENTS, AND CURSORS

Pseudo-classes style an element in a special state, such as :hover, :focus, :active, :checked, or :nth-child().

Pseudo-elements style a virtual part of an element, such as ::before, ::after, ::first-letter, and ::placeholder. Cursor styles can also signal interactivity by changing the pointer appearance.

These features work well together because they help build interactive, expressive interfaces without changing the HTML structure too much.
