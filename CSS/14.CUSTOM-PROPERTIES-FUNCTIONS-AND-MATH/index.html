CSS CUSTOM PROPERTIES, FUNCTIONS, AND MATH

Custom properties, often called CSS variables, let you store reusable values such as colors, spacing, and font sizes. They are created with names like --brand-color and used with var().

CSS functions like calc(), clamp(), min(), and max() help create flexible rules that respond to different screen sizes and layout needs.

These features are grouped together because they help stylesheets become more reusable, scalable, and easier to update.

This panel uses a CSS variable for color, calc() for spacing, and clamp() for responsive width.
