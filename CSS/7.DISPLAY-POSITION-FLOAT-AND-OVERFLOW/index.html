CSS DISPLAY, POSITION, FLOAT, AND OVERFLOW

The display property controls how an element behaves in the layout, such as block, inline, inline-block, flex, grid, or none.

The position property controls how an element is placed. Common values are static, relative, absolute, fixed, and sticky. Float is an older layout method that still appears in some projects, while overflow controls what happens when content does not fit inside a box.

These topics are grouped because they all affect how content is arranged, layered, or clipped inside the page.
