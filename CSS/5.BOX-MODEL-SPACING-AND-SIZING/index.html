CSS BOX MODEL, SPACING, AND SIZING

Every HTML element is treated like a rectangular box. The CSS box model includes content, padding, border, and margin.

Padding adds space inside the element, border wraps around the padding, and margin creates space outside the element. Width, height, min-width, max-width, and box-sizing control how large the box becomes.

Understanding the box model is one of the most important CSS skills because spacing and sizing problems often come from it.
