CSS BACKGROUNDS, BORDERS, AND SHADOWS

These properties control how surfaces and edges look. Backgrounds can use colors, images, gradients, positioning, repeating, and sizing rules.

Borders define the outline of an element, and shadows add depth with box-shadow or text-shadow. Rounded corners are created with border-radius.

These styles are often taught together because they strongly affect the visual personality of a component.
