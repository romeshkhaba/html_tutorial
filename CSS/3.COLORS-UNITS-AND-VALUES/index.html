CSS COLORS, UNITS, AND VALUES

CSS values describe what style should be applied. These values can be colors, measurements, keywords, percentages, functions, or other supported data types.

Colors can be written as named colors, hexadecimal values, RGB, RGBA, HSL, or HSLA. Measurements often use px, em, rem, %, vw, and vh.

Choosing the right unit matters because fixed units like px behave differently from flexible units like rem or percentages.
