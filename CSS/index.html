CSS LEARNING ROADMAP

This folder groups the main CSS topics into practical lessons so related ideas are taught together instead of being spread across too many tiny folders.

Lesson 1 introduces CSS basics and syntax. Lesson 2 explains selectors and specificity because choosing elements correctly is the foundation of styling.

Lesson 3 combines colors, units, and values. Lesson 4 groups text, fonts, and lists because these properties often work together when styling readable content.

Lesson 5 covers the box model, spacing, and sizing. Lesson 6 focuses on backgrounds, borders, and shadows because they shape the visual appearance of elements.

Lesson 7 teaches display, position, float, and overflow. Lesson 8 covers Flexbox, and Lesson 9 covers Grid because both are important modern layout systems.

Lesson 10 groups forms, tables, and user interface styling. Lesson 11 combines pseudo-classes, pseudo-elements, and cursors because they control interaction and generated content.

Lesson 12 teaches transitions, transforms, and animations. Lesson 13 explains responsive design and media queries so layouts can adapt to different screen sizes.

Lesson 14 combines custom properties, CSS functions, and math helpers like calc() and clamp(). Lesson 15 finishes with cascade, inheritance, and best practices because those rules affect every stylesheet you write.
