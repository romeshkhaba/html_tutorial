CSS FLEXBOX

Flexbox is a one-dimensional layout system that arranges items in a row or column. It is excellent for aligning content, spacing items evenly, and building responsive components.

Important flex properties include display: flex, justify-content, align-items, gap, flex-wrap, flex-grow, flex-shrink, and flex-basis.

Flexbox is especially useful for menus, cards, button groups, form rows, and any layout where items flow in a single direction.
