CSS SELECTORS AND SPECIFICITY

Selectors tell CSS which HTML elements should receive a style. Common selectors include element selectors, class selectors, id selectors, attribute selectors, descendant selectors, and child selectors.

Specificity is the rule that decides which style wins when more than one selector targets the same element. In general, id selectors are stronger than class selectors, and class selectors are stronger than element selectors.

Good CSS usually prefers reusable classes instead of styling everything with ids, because classes make designs easier to maintain and repeat.
