CSS BASICS AND SYNTAX

CSS stands for Cascading Style Sheets. It is used to control how HTML elements look, including their color, size, spacing, layout, and visual behavior.

A CSS rule is usually written as a selector followed by curly braces. Inside the braces, properties and values are written like this: selector { property: value; }.

CSS can be added in three common ways: inline with the style attribute, internal with a <style> tag, or external with a separate stylesheet file linked to the page.
