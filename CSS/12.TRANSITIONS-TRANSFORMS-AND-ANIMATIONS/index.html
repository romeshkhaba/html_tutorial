CSS TRANSITIONS, TRANSFORMS, AND ANIMATIONS

Transitions make property changes happen smoothly over time. Transforms change the shape or position of an element with functions like scale(), rotate(), translate(), and skew().

Animations use @keyframes to define multi-step motion that can loop, pause, delay, or run once. These tools are often taught together because they all add movement and visual change.

Good motion should support the design and improve feedback instead of distracting the user.
