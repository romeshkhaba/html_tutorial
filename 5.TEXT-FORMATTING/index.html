HTML TEXT FORMATTING

Text formatting tags help give meaning and emphasis to written content. They are used when plain paragraph text is not enough to show importance, stress, highlights, or special notation.

The most common text formatting elements are <strong>, <em>, <mark>, <small>, <sub>, <sup>, <code>, <q>, and <blockquote>.

These tags are useful because they improve meaning, accessibility, and readability instead of only changing appearance.

Strong text shows importance, emphasized text adds stress, and marked text highlights a key point.

Small text can be used for side notes, H2O uses subscript, and x2 uses superscript.

The <code> tag is helpful for code like <p>, and the q tag is used for short inline quotations.
