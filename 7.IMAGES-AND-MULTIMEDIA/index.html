HTML IMAGES AND MULTIMEDIA

HTML can display visual and audio content through elements such as <img>, <audio>, <video>, <source>, <track>, <figure>, and <figcaption>.

The <img> tag is used for images and should always include the alt attribute so the image still has meaning if it cannot be seen or loaded.

The <audio> and <video> elements can include controls for playback, while <picture> can serve different image files for different screen sizes.

<audio controls>
  <source src="song.mp3" type="audio/mpeg">
</audio>

<video controls width="320">
  <source src="movie.mp4" type="video/mp4">
</video>
