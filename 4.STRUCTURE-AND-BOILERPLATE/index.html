HTML STRUCTURE AND BOILERPLATE

Every HTML page begins with a basic structure that tells the browser what kind of document it is and where the metadata and visible content are placed.

The <!DOCTYPE html> line tells the browser to use modern HTML. The <html> tag wraps the whole page, the <head> stores metadata, and the <body> contains the content shown to users.

Comments written as <!-- comment --> help explain code for developers and are not displayed on the page.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>This is a complete HTML page.</p>
</body>
</html>
