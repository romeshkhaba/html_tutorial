HTML VOID ELEMENTS

Void elements are HTML tags that do not have a closing tag because they do not wrap content inside them. They perform a single job and end by themselves.

Common void elements include <br> for a line break, <hr> for a horizontal line, <img> for images, <input> for form fields, and <meta> for document information.
